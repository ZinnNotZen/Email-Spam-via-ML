# Email Spam Filter - Data Dictionary
This data set simulates am email data base to filter. 
Source: Kaggle Spam email Dataset: https://www.kaggle.com/datasets/jackksoncsie/spam-email-dataset
---

# emails.csv

Contains information about the origional dataset from Kaggle.

| Column | Description |
|------|-------------|
| text | The text content of the email messages, it includes the body of the emails along with any associated subject lines or headers. |
| spam | Contains binary labels to indicate whether an email is spam or not. "1" represents spam, while "0" represents not spam. |
